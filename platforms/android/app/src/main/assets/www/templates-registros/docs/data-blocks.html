Data blocks

Use standard Bootstrap grid classes with data-block cell to build your own layout. Each data block comes with optional components.

Data block structure

First grid layer is built with .data-block cells. Second one works inside of data blocks. You can remove <footer> and <header> if needed or turn whole block into more complex widget.

Each block header can be enchanted with buttons, dropdown, tabs, search bar or loading notification. Use .data-header-actions or .header-search class for proper styling. See the examples on this page.

<article class="col-sm-6">
	<div class="data-block">
		<header>
			<h2>Headline</h2>
		</header>
		<section>
			Content
		</section>
		<footer>
			Footer
		</footer>
	</div>
</article>

Data block widgets

  • Calendar

Use .widget-block class to turn data-block into a widget.

Follow the structure below so you can easily customize widget-block content.

<div class="col-sm-2">
	<a class="data-block widget-block">
		<section>
			<span class="widget-icon">Icon</span>
			<strong>Text</strong>
		</section>
	</a>
</div>

Data block plans

Data blocks with .plan-block class applied allows you to create blocks with more specific information. Simply follow the structure below.

<article class="col-sm-3">
	<div class="data-block plan-block">
		<section>
			<span>Icon</span>
			<h3>Headline</h3>
			<p>Text</p>
			<a class="btn">Button</a>
		</section>
	</div>
</article>

Data block with navigation

Most of the template components are combinable. You can take standard data block and standard tabular navigation to make secondary navigation inside the data block. See the following markup, class .data-block-nav is not required.

<article class="col-sm-6">
	<div class="data-block">
		<header>
			<h2>Block navigtaion</h2>
			<ul class="data-header-actions">
				<li><a href="#subone">Nav 1</a></li>
				<li><a href="#subtwo">Nav 2</a></li>
			</ul>
		</header>
		<div class="data-block-nav tab-content">
			<div class="tab-pane" id="subone">
				<ul>
					<li><a href="#">Link</a></li>
					...
				</ul>
			</div>
			<div class="tab-pane" id="subtwo">
				<ul>
					<li><a href="#">Link</a></li>
					...
				</ul>
			</div>
		</div>
		<section>
			...
		</section>
	</div>
</article>
