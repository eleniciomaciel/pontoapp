Forms

All form elements are styled according to Twitter Bootstrap syntax. I you are not familiar with bootstrap framework you should check their official documentation.

Sangoma template uses several form handling plugins. Check them out on plugins page.

Icons

Template includes 4 icon sets including standad Bootstrap iconfont Glypicons, iconfont Elusives which is used as default, iconfont Font Awesome and image icons FamFamFam. Take a look at propriate LESS file and icon set homepage for correct use and naming. All icon sets are included, just uncomment icon set you want in style.less, edit style where necessary and recompile you CSS.
